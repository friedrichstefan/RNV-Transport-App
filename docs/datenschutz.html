Kurzfassung: ÖPNV Mannheim ist ein nicht-kommerzielles Studentenprojekt. Die App sammelt keine personenbezogenen Daten, enthält kein Tracking und gibt keinerlei Informationen an Dritte weiter.

1. Allgemeines

Diese Datenschutzerklärung gilt für die iOS-App „ÖPNV Mannheim" sowie die zugehörige Website. Die App ist ein unabhängiges, nicht-kommerzielles Studentenprojekt ohne Verbindung zur rnv GmbH oder anderen Verkehrsbetrieben.

Wir nehmen den Schutz deiner Daten ernst. Diese Erklärung informiert dich darüber, welche Daten verarbeitet werden und warum.

Standortdaten

Die App kann deinen Standort verwenden, um nahegelegene Haltestellen automatisch zu finden. Der Standortzugriff ist optional und kann jederzeit in den iOS-Einstellungen unter Einstellungen → Datenschutz → Ortungsdienste → ÖPNV Mannheim deaktiviert werden.

Standortdaten werden ausschließlich lokal auf deinem Gerät verarbeitet, um die API-Anfrage zu formulieren. Sie werden weder gespeichert noch an Dritte weitergegeben.

Netzwerkanfragen

Anfragen für Echtzeit-Fahrplandaten werden direkt an die öffentliche GraphQL-API des RNV gesendet. Es existiert kein eigener Server. Die App überträgt dabei keine personenbezogenen Daten – lediglich die gesuchte Haltestelle oder der Standortradius werden als Parameter übermittelt.

Lokale Datenspeicherung

Die App speichert folgende Daten ausschließlich lokal auf deinem Gerät (via UserDefaults), ohne Übertragung ins Internet:

· Zuletzt genutzte Haltestellen
· App-Einstellungen (Filter, Suchradius)
· Daten für laufende Live Activities (Dynamic Island)

Du kannst alle gespeicherten Daten jederzeit über Einstellungen → Cache leeren in der App löschen.

Drittanbieter & Tracking

Die App enthält keine Analytics-SDKs, Werbenetzwerke, Crash-Reporter oder sonstige Drittanbieter-Dienste, die Daten erfassen oder weitergeben.

2. Website

Diese Website wird über GitHub Pages gehostet. GitHub kann dabei technische Zugriffsdaten (IP-Adresse, Zeitstempel) in Server-Logs erfassen. Weitere Informationen findest du in der Datenschutzerklärung von GitHub.

Diese Website verwendet keine Cookies, kein Tracking, keine Analytics und keine eingebetteten Drittanbieter-Inhalte.

3. Deine Rechte

Da die App keine personenbezogenen Daten erhebt oder speichert, entfällt der Großteil der üblichen DSGVO-Rechte (Auskunft, Berichtigung, Löschung) mangels Datenerhebung.

Bei Fragen oder Anliegen zum Datenschutz kannst du uns über GitHub Issues kontaktieren.
